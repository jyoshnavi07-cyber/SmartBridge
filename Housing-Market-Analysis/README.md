# Visualizing Housing Market Trends: An Analysis of Sale Prices and Features using Tableau

## Project Overview

This project focuses on analyzing residential housing data to identify the key factors that influence house sale prices. Using Tableau, interactive dashboards and visualizations were developed to uncover trends related to property features such as living area, overall quality, neighborhood, year built, garage area, and sale price.

The project demonstrates how data visualization can help users better understand housing market trends and support data-driven decision-making.

---

## Objectives

* Analyze housing market trends using Tableau.
* Explore the relationship between property features and sale prices.
* Identify the factors that most influence housing prices.
* Build interactive dashboards for easy data exploration.
* Present insights through visual storytelling.

## Project Workflow

### 1. Data Preparation

* Imported the housing dataset into Tableau.
* Verified data types and field formats.
* Checked for missing or inconsistent values.
* Prepared the data for visualization.

### 2. Data Visualization

Created multiple worksheets to analyze different aspects of the housing market.

### 3. Dashboard Development

Combined individual worksheets into an interactive dashboard with filters and KPI cards.

### 4. Analysis

Interpreted the visualizations to identify housing market trends and key insights.

## Key Insights

* Houses with larger living areas generally have higher sale prices.
* Overall quality has a strong impact on property value.
* Newer homes often command higher market prices.
* Property prices vary significantly across neighborhoods.
* Interactive dashboards make housing market exploration simple and effective.
